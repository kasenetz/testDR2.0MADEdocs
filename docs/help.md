# Help and Resources

Please direct any inquiries regarding HBCD-MADE to <eegdata@umd.edu>. 

Questions about the HBCD EEG data release can be sent by creating a help desk ticket on the Lasso platform - see instructions [here](https://docs.hbcdstudy.org/latest/help/reportissue/).
